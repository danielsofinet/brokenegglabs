# Session 2026-06-17 — Full overhaul of brokenegglabs.com

Reworked Daniel's old (2022) studio site. All 7 of his listed bugs addressed.

## What was done
1. **One-site navigation + URL always `/`** — built a clean client-side router in
   `js/bke.js`. Intercepts internal `.html` links, fetches + swaps `<main>` content,
   never touches the URL bar. Replaced Barba.js (which navigated by real URLs).
2. **Circular page transition** — reproduced the brand's circle reveal with a CSS
   clip-path overlay (`.bke-trans` in `bke.css`) expanding from the click point.
3. **Process slider fixed** — the dead GSAP pin-scroll on the About page is now a
   drag-to-scroll horizontal strip (pointer drag + wheel + touch).
4. **Videos** — localized to `assets/video/`, compressed with ffmpeg (no audio,
   faststart), autoplay-on-view via IntersectionObserver, `preload="metadata"` for
   progressive loading.
5. **Dates** — copyright/"current" 2022 → 2026. (Portfolio project years like
   "Minot Vin 2022" left as-is — those are historical completion dates.)
6. **Contact** — real modal form (`bke.css`/`bke.js`) wired to Web3Forms →
   daniel.sofinet@gmail.com. Removed all `hello@brokenegg.labs`; CTAs now say
   "contact us here". CONTACT nav still opens the contact PAGE; on-page CTAs open
   the modal. Phone fixed to `06 73 67 22 00`. Hero placeholder copy "Click goes
   here" → "Got a project in mind?".
7. **Auto-hide header** — folds up on scroll down, drops on scroll up, on every page.

## Cleanup
- Removed ~628KB of dead JS: `main.js` (526KB), `jquery.min.js`, `jquery-migrate.min.js`,
  `wp-embed.min.js`. Only `js/bke.js` (12KB) + SproutVideo player API remain.
- `bke.css` neutralizes the old ASScroll smooth-scroll + WordPress preloader (which
  the deleted main.js used to drive) so native scrolling works and pages aren't blank.

## Key discovery (important for future debugging)
The blank-page / frozen-scroll problems were NOT the router — they were the orphaned
ASScroll library + intro preloader. `.ready` (added by old JS) also set
`overflow:hidden; height:100vh`. `.pre-load` is a fixed dark full-screen overlay the
old JS removed after load. `bke.css` now forces all of this off.

## Tested in-browser (Chrome DevTools MCP)
Home, About (pr.html), Work (cs.html), Contact, and a project page (minot-vin): render,
scroll, nav-fold, circle transition, slider drag, video autoplay-on-view, contact modal,
and injected-content asset resolution all verified. Console clean (0 errors).
